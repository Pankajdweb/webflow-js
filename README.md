Webflow JS static site

This is a simple static site ready to be deployed on Netlify.

## Deploying to Netlify

1. Push this repository to GitHub, GitLab, or Bitbucket.
2. In the Netlify dashboard, create a new site from your git repo.
3. When asked for build settings:
   - Build command: leave empty (no build step).
   - Publish directory: `.` (project root).
4. Save and deploy.

Netlify will serve the `index.html` file from the project root.
